📌 Fitness Planner

A simple and interactive web-based fitness planning application that helps users:

Track daily exercises

Calculate calories burned

Get suggested food intake

Maintain a balanced fitness routine

Developed by Varun AL

📋 Features

Select different types of exercises

Automatic calorie calculation

Food recommendations based on calories burned

Clean and simple user interface

Fully responsive design

Cloud deployment

💡 How It Works

1. User selects an exercise type.

2. User enters duration (in minutes).

3. Application calculates calories burned.

4. Suggested food intake is displayed accordingly.

🖥 How to Run Locally
